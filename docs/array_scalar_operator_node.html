arrayScalarOp

This node calculates a binary operation between each value in the input array and the scalar value and outputs the results.

Example

node = cmds.createNode('arrayScalarOp')
cmds.setAttr(node + ".input", range(10), type="doubleArray")
cmds.setAttr(node + ".operation", 3)
cmds.getAttr(node + ".output")
# Result [0.0, 3.0, 6.0, 9.0, 12.0, 15.0, 18.0, 21.0, 24.0, 27.0, 30.0] #
