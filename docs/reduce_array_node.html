reduceArray

This node applies an accumulating function to the values of the input array and outputs a single value.

Example

node = cmds.createNode('reduceArray')
cmds.setAttr(node + ".input", range(10), type="doubleArray")
cmds.setAttr(node + ".operation", 1)
cmds.getAttr(node + ".output")
