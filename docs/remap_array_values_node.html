remapArray

This node remaps the values of the input array using a gradient.

Example

node = cmds.createNode('remapArrayValues')
cmds.setAttr(node + ".input[0]", range(11), type="doubleArray")
cmds.setAttr(node + ".inputMin", 0.0)
cmds.setAttr(node + ".inputMax", 10.0)
cmds.setAttr(node + ".values[0].values_Interp", 2)
cmds.getAttr(node + ".output")
# Result: [0.0, 0.024, 0.095, 0.206, 0.345, 0.5, 0.655, 0.794, 0.905, 0.976, 1.0] #
