arrayMultiOp

This node accumulates the values in the input arrays to a function and outputs the results.

Example

node = cmds.createNode('arrayMultiOp')
cmds.setAttr(node + ".input[0]", range(10), type="doubleArray")
cmds.setAttr(node + ".input[1]", range(10), type="doubleArray")
cmds.setAttr(node + ".input[2]", range(10), type="doubleArray")
cmds.setAttr(node + ".operation", 1)
cmds.getAttr(node + ".output")
# Result [0.0, 3.0, 6.0, 9.0, 12.0, 15.0, 18.0, 21.0, 24.0, 27.0, 30.0] #
